# collimator
This is a collection of scripts to design 3d printed collimators.

## Steps to design a (Soller) collimator
1. create grid
1. create 3d model
1. validate 3d model

## Software dependencies
- python
- [FreeCAD](https://www.freecadweb.org)
- [OpenSCAD](https://www.openscad.org)
